Privacy Policy

Last updated: March 2026 · Effective: March 2026

Plain-English summary: TekkieStack is designed for children. We collect the bare minimum data needed to make it work. Nothing is sold. Nothing is shared with advertisers. All learner data stays on your own device — we never see it.

Contents
1. Who We Are 2. Data We Store Locally 3. Data Sent to Third Parties 4. Children's Privacy 5. Your Rights 6. Contact

1. Who We Are

TekkieStack is a product of Aperintel. "We", "us", or "our" refers to Aperintel. Please use our contact form in the Help & Support section.

2. Data We Store Locally on Your Device

All of the following is stored only on the device you use — in the browser's IndexedDB storage. It is never transmitted to Aperintel Ltd or any server.

  • First name and chosen avatar emoji
  • Year group (e.g. Year 5)
  • A hashed PIN (we store a one-way hash, never the PIN itself)
  • Learning progress — lessons completed, phases, XP points, badges
  • Code projects saved in the editor
  • Typing stats — words per minute, session count
  • Streak and daily challenge history

We do not collect: full names, email addresses, phone numbers, school names, location, photographs, biometric data, or any behavioural tracking data.

3. Data Sent to Third Parties

Anthropic (AI Lab — Y5+ only, disabled by default)

When the optional online AI Lab is enabled for Year 5+ learners, the text of the coding question is sent to Anthropic's API. Before sending, our safety layer strips all personal information. The learner's name, year group, device ID, and school are never included in API calls. Anthropic's privacy policy: anthropic.com/privacy

EmailJS (Support contact form)

When you submit the support form, your message and optional name are routed securely to the Aperintel support team. No profile data or learning progress is included. EmailJS privacy policy: emailjs.com/legal/privacy-policy

Google Fonts

The app loads fonts from fonts.googleapis.com. Google's servers receive a standard HTTP request that includes your IP address and browser type — this is normal for any website that uses Google Fonts. No cookies are set. You can disable Google Fonts on the Cookie & Storage Policy page to use system fonts instead.

4. Children's Privacy

TekkieStack is built specifically for children. Children's data protection is a foundational design principle, not an afterthought.

  • UK GDPR & Children's Code: We comply with the UK GDPR and the ICO's Age Appropriate Design Code (Children's Code).
  • Data minimisation: We collect only what is necessary for the product to function.
  • No profiling: We do not profile children, run advertising, or make automated decisions about learners.
  • No social features: There are no public profiles, no peer comparisons, and no sharing of learner data between devices.
  • AI safety: Year 3–4 children cannot access the online AI at all. Year 5+ can only access it when a parent or teacher enables it, and personal information is always stripped before queries are sent.
  • COPPA: TekkieStack does not collect personal information from children on any server, so COPPA's operator requirements do not apply in Release 1.

5. Your Rights

Because all learner data is stored locally on your device, you are the data controller of that information. You can exercise full control at any time:

  • Access: All data is visible within the Guardian View feature.
  • Deletion: Use the "Clear All My TekkieStack Data" button on the Cookie & Storage Policy page, or clear site data in your browser settings. Deletion is immediate and complete.
  • Portability: The Portfolio Generator exports your data as a downloadable HTML file.

For data sent to third parties (support emails), please use the Help & Support contact form.

Cookie & Storage Policy

Last updated: March 2026 · Effective: March 2026

Plain-English summary: We use zero cookies. We use browser storage to save your learning progress locally on your device. You can delete everything at any time with one button. No tracking, no ads, no data sent to us.

Contents
1. No Cookies 2. Browser Storage We Use 3. Third-Party Storage 4. Your Controls 5. Browser-Specific Instructions

1. We Use No Cookies

TekkieStack does not set any cookie files on your device — not session cookies, not persistent cookies, not tracking cookies, not advertising cookies. If you inspect your browser's cookie store for this site, it will be empty.

The "cookie banner" you see when you first visit is required by UK and EU law for any site that uses browser storage. We show it to be transparent about the storage we do use, even though it is not technically cookies.

2. Browser Storage We Use

IndexedDB — Learning data

Purpose: stores your learner profiles, lesson progress, XP, badges, code projects, typing stats, and automatic backups. This is the primary data store for TekkieStack.

Database name: tekkiestack · Schema version 5 · Stays on your device · Never transmitted

localStorage — Preferences

Purpose: stores a small number of lightweight flags — whether you have accepted this cookie notice (ts_cookie_ok), your device ID used for PIN hashing (ts_device_id), and whether you've dismissed the app install prompt. Contains no personal information about the learner.

Cache API — Offline shell

Purpose: caches the app's HTML, CSS, JavaScript, and fonts so that TekkieStack works with no internet connection after the first load. Cache name: ts-v2. Contains only application files — no user data.

3. Third-Party Storage

TekkieStack loads Google Fonts from fonts.googleapis.com. Google may set its own cache entries in your browser as part of the font delivery. We have no control over Google's caching behaviour. You can disable Google Fonts on this page (see controls below) to prevent any Google network requests.

When the optional online AI Lab is in use, the Anthropic API is called directly from your browser. Anthropic does not set any storage on your device.

4. Your Controls

🔤 Google Fonts
Disable to prevent Google network requests. App uses system fonts instead.
🤖 Online AI (Y5+)
Disabled by default. Enable per-profile by a parent or teacher.

Permanently deletes all profiles, projects, progress, and backups on this device. Cannot be undone.

5. How to Clear Data in Your Browser

You can also clear TekkieStack's storage directly from your browser:

  • Chrome / Edge: Settings → Privacy and security → Site settings → View permissions and data stored across sites → Search for this site → Delete
  • Safari (Mac): Develop menu → Web Application → IndexedDB; or Settings → Privacy → Manage Website Data
  • Safari (iPhone/iPad): Settings → Safari → Advanced → Website Data → Edit → find this site → Delete
  • Firefox: Settings → Privacy & Security → Cookies and Site Data → Manage Data → search for this site → Remove
